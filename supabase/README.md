# OmniGuard Backend

This folder contains the Supabase backend configuration and Edge Functions for OmniGuard.

## Edge Functions
- **analyze-threat**: Analyzes security threats using the AI model.
- **chat**: Handles chatbot interactions and processes queries.

## Local Development
To run or deploy these Edge Functions locally, make sure you have the Supabase CLI installed, then run:
```bash
supabase functions serve
```
